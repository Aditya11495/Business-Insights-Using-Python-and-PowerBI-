# End-to-End Retail Sales Analysis

## Project Overview

This project demonstrates an end-to-end Data Analytics workflow using Python and Power BI. The objective is to analyze retail sales data, identify key business trends, evaluate product and regional performance, and generate actionable business insights.

The project covers data understanding, data quality assessment, exploratory data analysis (EDA), KPI generation, visualization, and business recommendations.


## Business Problem

ABC Retail wants to understand:

* Which regions generate the highest revenue?
* Which products contribute the most sales and profit?
* Which states are underperforming?
* How profitable is the business?
* What insights can help management improve decision-making?



## Dataset Information

The dataset contains 1000 retail sales transactions with the following fields:

* Order_ID
* Order_Date
* Region
* State
* Customer_ID
* Category
* Product
* Sales
* Quantity
* Discount
* Profit

## Key Findings

### Regional Analysis

* East region generated the highest sales revenue.
* South region ranked second in total sales contribution.

### State Analysis

* West Bengal was the highest-performing state by sales.
* Gujarat and Punjab were among the lower-performing states.

### Product Analysis

* Notebook generated the highest sales.
* Notebook also generated the highest profit.

### Business Performance

* Total Sales: 25.45M
* Total Profit: 3.12M
* Profit Margin: 12.25%
* Average Discount: 14.53%



## Business Recommendations

1. Focus marketing efforts on high-performing regions such as East.
2. Increase inventory planning for top-selling products like Notebook.
3. Investigate lower-performing states to identify growth opportunities.
4. Monitor discount strategies to maintain healthy profit margins.
5. Use regional performance insights to optimize sales strategies.
